# AvatarGroup

A container component that displays multiple avatars in a stacked horizontal layout with automatic overflow handling. When the number of avatars exceeds the maximum limit, it shows a "+N" indicator.

## Installation & Integration

### Import

```tsx
import { AvatarGroup, Avatar } from "dandy-ui";
```

## Props

| Prop      | Type                                | Default | Description                                              |
| --------- | ----------------------------------- | ------- | -------------------------------------------------------- |
| max       | `number`                            | -       | Maximum avatars to display before showing "+N" indicator |
| size      | `xs \| sm \| md \| lg \| xl \| 2xl` | `md`    | Size applied to all child avatars                        |
| spacing   | `sm \| md \| lg`                    | `md`    | Spacing between stacked avatars (negative margin)        |
| children  | `React.ReactNode`                   | -       | Avatar elements to display                               |
| className | `string`                            | -       | Additional CSS classes                                   |

Inherits all standard HTML `div` attributes (`onClick`, `role`, etc.).

## Accessibility

- Uses semantic `role="group"` with descriptive `aria-label` indicating total member count
- Overflow indicator includes `aria-label` for remaining members count
- All child avatars maintain their individual accessibility features
- Screen reader compatible with proper member count announcements
- Keyboard navigable if avatars have interactive elements
- WCAG 2.1 Level AA compliant
