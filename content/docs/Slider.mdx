# Slider

A mobile-first range input component for selecting numeric values within a defined range. Features smooth drag interactions, keyboard navigation, and full accessibility support.

## Installation & Integration

### Import

```tsx
import { Slider } from "dandy-ui";
```

## Props

| Prop       | Type                                                                    | Default   | Description                          |
| ---------- | ----------------------------------------------------------------------- | --------- | ------------------------------------ |
| value      | `number`                                                                | `0`       | Current slider value (controlled)    |
| onChange   | `(value: number) => void`                                               | -         | Callback fired when value changes    |
| min        | `number`                                                                | `0`       | Minimum value                        |
| max        | `number`                                                                | `100`     | Maximum value                        |
| step       | `number`                                                                | `1`       | Step increment                       |
| variant    | `primary \| secondary \| accent \| success \| warning \| error \| info` | `primary` | Visual style variant                 |
| size       | `sm \| md \| lg`                                                        | `md`      | Component size                       |
| label      | `string`                                                                | -         | Label displayed above slider         |
| showValue  | `boolean`                                                               | `false`   | Whether to show current value        |
| disabled   | `boolean`                                                               | `false`   | Disables the slider                  |
| aria-label | `string`                                                                | -         | Custom aria-label for screen readers |
| name       | `string`                                                                | -         | Name attribute for form integration  |
| className  | `string`                                                                | -         | Additional CSS classes               |

Inherits all standard HTML `div` attributes (`onClick`, `onMouseEnter`, etc.).

## Accessibility

- Keyboard navigable with Arrow keys (left/right/up/down for step, Page Up/Down for 10Ã— step, Home/End for min/max)
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
- Touch-friendly sizing with minimum 44x44px thumb targets
- Screen reader compatible with proper ARIA attributes (`role="slider"`, `aria-valuemin`, `aria-valuemax`, `aria-valuenow`)
- Disabled state properly communicated to assistive technologies
- Mouse, touch, and keyboard interaction support
