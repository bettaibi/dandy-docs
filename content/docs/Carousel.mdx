# Carousel

A composable carousel component built on Embla Carousel, featuring touch/swipe gestures, keyboard navigation, and extensive customization options.

## Installation & Integration

### Import

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from "dandy-ui";
```

### Peer Dependencies

This component requires the following peer dependency:

```bash
npm install embla-carousel-react
```

## Props

### Carousel Props

| Prop        | Type                         | Default      | Description                                   |
| ----------- | ---------------------------- | ------------ | --------------------------------------------- |
| orientation | `horizontal \| vertical`     | `horizontal` | Carousel scroll direction                     |
| opts        | `CarouselOptions`            | -            | Embla Carousel options object                 |
| plugins     | `CarouselPlugin`             | -            | Embla Carousel plugins                        |
| setApi      | `(api: CarouselApi) => void` | -            | Callback to receive the carousel API instance |
| hideArrows  | `boolean`                    | `false`      | Hide navigation arrows                        |
| className   | `string`                     | -            | Additional CSS classes                        |

Inherits all standard HTML div attributes.

### CarouselItem Props

| Prop      | Type     | Default | Description            |
| --------- | -------- | ------- | ---------------------- |
| className | `string` | -       | Additional CSS classes |

Inherits all standard HTML div attributes.

### CarouselPrevious / CarouselNext Props

| Prop       | Type                        | Default   | Description             |
| ---------- | --------------------------- | --------- | ----------------------- |
| variant    | `default \| primary \| ...` | `primary` | Button color variant    |
| appearance | `filled \| outlined`        | `filled`  | Button appearance style |
| size       | `sm \| md \| lg`            | `md`      | Button size             |
| className  | `string`                    | -         | Additional CSS classes  |

Inherits all IconButton component props.

## Accessibility

- Keyboard navigable with Arrow keys (Left/Right for horizontal, Up/Down for vertical)
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
- Proper ARIA attributes (`role="region"`, `aria-roledescription="carousel"`, `aria-roledescription="slide"`)
- Navigation buttons have descriptive `aria-label` attributes
- Touch and swipe gestures supported on mobile devices
- Navigation buttons automatically disabled when at start/end (unless loop is enabled)

## Component API

| Export           | Type                              | Description                   |
| ---------------- | --------------------------------- | ----------------------------- |
| Carousel         | `React.ForwardRefExoticComponent` | Main carousel container       |
| CarouselContent  | `React.FC`                        | Viewport and slides container |
| CarouselItem     | `React.FC`                        | Individual slide wrapper      |
| CarouselPrevious | `React.FC`                        | Previous navigation button    |
| CarouselNext     | `React.FC`                        | Next navigation button        |
| CarouselApi      | `type`                            | Embla Carousel API type       |
