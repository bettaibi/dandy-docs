# Alert

A versatile notification component for displaying important messages to users with semantic color variants, optional icons, and flexible content layouts.

## Installation & Integration

### Import

```tsx
import { Alert } from "dandy-ui";
```

## Props

| Prop          | Type                                             | Default   | Description                                      |
| ------------- | ------------------------------------------------ | --------- | ------------------------------------------------ |
| variant       | `default \| success \| warning \| error \| info` | `default` | Visual style variant with semantic colors        |
| title         | `React.ReactNode`                                | -         | Optional heading/title text                      |
| description   | `React.ReactNode`                                | -         | Optional description (falls back to children)    |
| showIcon      | `boolean`                                        | `true`    | Show the leading icon                            |
| icon          | `React.ReactNode`                                | -         | Provide a custom icon (defaults to variant icon) |
| rounded       | `sm \| md \| lg \| xl \| full \| none`           | `lg`      | Border radius size                               |
| elevation     | `none \| sm \| md \| lg \| xl`                   | `none`    | Shadow elevation level                           |
| customContent | `boolean`                                        | `false`   | Whether to use custom content layout             |
| className     | `string`                                         | -         | Additional CSS classes                           |

Inherits all standard HTML `div` attributes (`onClick`, `role`, etc.).

## Accessibility

- Uses semantic `role="alert"` by default for screen reader announcements
- Icons are marked `aria-hidden="true"` to avoid redundant announcements
- Focus visible with ring indicator
- High contrast border and background colors for WCAG compliance
- Semantic color variants for different message types
