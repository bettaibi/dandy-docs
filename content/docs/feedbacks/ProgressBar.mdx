# ProgressBar

Visual indicator component for displaying completion progress, loading states, or task status with support for labels, percentages, and multiple visual variants.

## Installation & Integration

### Import


## Props

| Prop           | Type                                                                               | Default   | Description                            |
| -------------- | ---------------------------------------------------------------------------------- | --------- | -------------------------------------- |
| value          | `number`                                                                           | -         | Current progress value (required)      |
| max            | `number`                                                                           | `100`     | Maximum value                          |
| variant        | `default \| primary \| secondary \| accent \| success \| warning \| error \| info` | `default` | Visual variant / status indicator      |
| size           | `sm \| md \| lg \| xl`                                                             | `md`      | Height of the progress bar             |
| label          | `string`                                                                           | `""`      | Optional label displayed above the bar |
| showPercentage | `boolean`                                                                          | `false`   | Whether to show percentage text        |
| aria-label     | `string`                                                                           | -         | Custom aria-label for screen readers   |
| className      | `string`                                                                           | -         | Additional CSS classes                 |

Inherits all standard HTML div attributes (`onClick`, `style`, etc.).

## Accessibility

- Uses semantic `role="progressbar"` with proper ARIA attributes
- Includes `aria-valuemin`, `aria-valuemax`, and `aria-valuenow` for screen readers
- Automatic aria-label based on label prop or defaults to "Progress"
- Respects `prefers-reduced-motion` for animation preferences
- Values are automatically clamped to valid range (0 to max)
- WCAG 2.1 Level AA compliant
- Screen reader compatible with progress announcements
