# Button

A flexible button component that supports multiple appearances, variants, loading states, and customizable slots for icons and badges.

## Installation & Integration

### Import

```tsx
import { Button } from "dandy-ui";
```

## Props

| Prop       | Type                                                                   | Default   | Description                                     |
| ---------- | ---------------------------------------------------------------------- | --------- | ----------------------------------------------- |
| appearance | `filled \| outlined`                                                   | `filled`  | Button appearance style                         |
| variant    | `primary \| secondary \| ghost \| success \| warning \| error \| info` | `primary` | Button color variant                            |
| size       | `sm \| md \| lg`                                                       | `md`      | Button size                                     |
| rounded    | `sm \| md \| lg \| full \| none`                                       | -         | Border radius (defaults to theme button radius) |
| loading    | `boolean`                                                              | `false`   | Shows loading spinner and disables the button   |
| leftSlot   | `React.ReactNode`                                                      | -         | Content rendered before button text             |
| rightSlot  | `React.ReactNode`                                                      | -         | Content rendered after button text              |
| disabled   | `boolean`                                                              | `false`   | Disables the button                             |
| className  | `string`                                                               | -         | Additional CSS classes                          |
| children   | `React.ReactNode`                                                      | -         | Button content                                  |

Inherits all standard HTML button attributes (`onClick`, `type`, `form`, etc.).

## Accessibility

- Keyboard navigable with Tab and Enter/Space keys
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
- Minimum 44x44px touch targets for mobile (height: 44px on all sizes)
- Automatically disabled during loading state
- Screen reader compatible with proper semantic button element
