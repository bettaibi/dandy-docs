# IconButton

A versatile icon button component with multiple variants, sizes, and appearance options. Ideal for actions represented by icons, toolbar buttons, and compact UI controls.

## Installation & Integration

### Import

```tsx
import { IconButton } from "dandy-ui";
```

## Props

| Prop       | Type                                                                   | Default   | Description             |
| ---------- | ---------------------------------------------------------------------- | --------- | ----------------------- |
| size       | `sm \| md \| lg`                                                       | `md`      | Button size             |
| variant    | `primary \| secondary \| ghost \| success \| warning \| error \| info` | `primary` | Color variant           |
| appearance | `filled \| outlined`                                                   | `filled`  | Visual appearance style |
| rounded    | `sm \| md \| lg \| full \| none`                                       | -         | Border radius override  |
| elevation  | `0 \| 1 \| 2 \| 3`                                                     | `0`       | Shadow elevation level  |
| disabled   | `boolean`                                                              | `false`   | Disables the button     |
| children   | `React.ReactNode`                                                      | -         | Icon element to render  |
| className  | `string`                                                               | -         | Additional CSS classes  |

Inherits all standard HTML button attributes (`onClick`, `type`, `aria-label`, etc.).

## Accessibility

- Keyboard navigable with Tab and Enter/Space keys
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
- Minimum 32x32px touch target on small size (sm: 32px, md: 40px, lg: 48px)
- Disabled state prevents interaction and reduces opacity
- Always include `aria-label` for screen reader accessibility since icons don't provide text context

```tsx
<IconButton aria-label="Close dialog">
  <XIcon />
</IconButton>
```
