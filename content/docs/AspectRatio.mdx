# AspectRatio

A container component that maintains a consistent width-to-height ratio, perfect for responsive images, videos, and embedded content.

## Installation & Integration

### Import

```tsx
import { AspectRatio } from "dandy-ui";
```

## Props

| Prop      | Type     | Default | Description                                             |
| --------- | -------- | ------- | ------------------------------------------------------- |
| ratio     | `number` | `1`     | Desired aspect ratio (e.g., 16/9, 4/3, 1)               |
| className | `string` | -       | Additional CSS classes                                  |
| style     | `object` | -       | Inline styles (merged with internal aspect ratio style) |

Inherits all standard HTML `div` attributes (`onClick`, `id`, etc.).

## Accessibility

- Uses semantic HTML structure with proper nesting
- Maintains aspect ratio across all screen sizes and zoom levels
- Works seamlessly with screen readers
- Supports keyboard navigation for interactive content
- Ensures proper focus management for embedded content

| Export           | Type                              | Description                        |
| ---------------- | --------------------------------- | ---------------------------------- |
