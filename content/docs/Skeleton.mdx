# Skeleton

Displays animated loading placeholder for content areas while data is being fetched. Features smooth pulse animation, customizable dimensions, and dark mode support.

## Installation & Integration

### Import

```tsx
import { Skeleton } from "dandy-ui";
```

## Props

| Prop      | Type                                             | Default | Description                |
| --------- | ------------------------------------------------ | ------- | -------------------------- |
| width     | `string`                                         | -       | Fixed width (CSS value)    |
| minWidth  | `string`                                         | -       | Minimum width (CSS value)  |
| maxWidth  | `string`                                         | -       | Maximum width (CSS value)  |
| height    | `string`                                         | -       | Fixed height (CSS value)   |
| minHeight | `string`                                         | -       | Minimum height (CSS value) |
| maxHeight | `string`                                         | -       | Maximum height (CSS value) |
| rounded   | `none \| sm \| md \| lg \| xl \| full \| string` | `md`    | Border radius control      |
| className | `string`                                         | -       | Additional CSS classes     |
| style     | `React.CSSProperties`                            | -       | Inline styles              |

Inherits all standard HTML div attributes.

## Accessibility

- Uses `aria-live="polite"` for screen reader announcements
- `aria-busy="true"` indicates loading state
- Smooth pulse animation provides visual loading feedback
- Dark mode support with proper contrast

## Component API

| Export        | Type                              | Description                        |
| ------------- | --------------------------------- | ---------------------------------- |
| Skeleton      | `React.ForwardRefExoticComponent` | Main component with ref forwarding |
| SkeletonProps | `interface`                       | Component props interface          |
