# Code

A versatile code display component supporting both inline snippets and multi-line code blocks. Includes built-in syntax highlighting for common languages, optional line numbers, and multiple color variants for contextual emphasis.

## Installation & Integration

### Import

```tsx
import { Code } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop            | Type                                                                                    | Default     | Description                             |
| --------------- | --------------------------------------------------------------------------------------- | ----------- | --------------------------------------- |
| size            | `xs \| sm \| base \| lg`                                                               | `sm`        | Text size of the code                   |
| variant         | `default \| primary \| secondary \| success \| warning \| error \| info`                | `default`   | Color variant                           |
| block           | `boolean`                                                                               | `false`     | Render as a full-width code block       |
| language        | `javascript \| typescript \| jsx \| tsx \| html \| css \| json \| python \| sql \| bash \| shell \| plaintext` | -  | Language for syntax highlighting        |
| showLineNumbers | `boolean`                                                                               | `false`     | Show line numbers (block mode only)     |
| className       | `string`                                                                                | -           | Additional CSS classes                  |

Inherits all standard HTML `code` element attributes.

## Accessibility

- Uses semantic `<code>` element for proper code identification
- Monospace font ensures consistent character width for readability
- Block mode supports horizontal scrolling with `overflow-x: auto`
- Line numbers are hidden from screen readers with `aria-hidden="true"`
- Color variants maintain readable contrast in both light and dark modes
