# Text

A flexible, polymorphic text component for rendering body copy, labels, captions, and inline content. Supports multiple sizes, weights, color variants, and text decorations with the ability to render as different HTML elements.

## Installation & Integration

### Import

```tsx
import { Text } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop        | Type                                                        | Default     | Description                                  |
| ----------- | ----------------------------------------------------------- | ----------- | -------------------------------------------- |
| size        | `xs \| sm \| base \| lg \| xl \| 2xl \| 3xl \| 4xl`       | `base`      | Text size variant                            |
| weight      | `normal \| medium \| semibold \| bold`                      | `normal`    | Text font weight                             |
| variant     | `default \| muted \| success \| warning \| error \| info`  | `default`   | Text color variant                           |
| as          | `p \| span \| div`                                         | `p`         | HTML element to render                       |
| lineThrough | `boolean`                                                   | `false`     | Apply line-through text decoration           |
| uppercase   | `boolean`                                                   | `false`     | Transform text to uppercase                  |
| underline   | `boolean`                                                   | `false`     | Apply underline text decoration              |
| truncate    | `boolean`                                                   | `false`     | Truncate text with ellipsis on overflow      |
| trim        | `boolean`                                                   | `false`     | Trim leading and trailing whitespace         |
| wrap        | `boolean`                                                   | `false`     | Prevent text wrapping                        |
| className   | `string`                                                    | -           | Additional CSS classes                       |

Inherits all standard HTML `p` element attributes.

## Accessibility

- Renders semantic HTML elements (`p`, `span`, `div`) based on the `as` prop
- Color variants maintain sufficient contrast ratios for WCAG 2.1 Level AA
- Text sizes remain readable across all viewport sizes
- Supports screen readers with proper semantic structure
