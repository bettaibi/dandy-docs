# Kbd

A keyboard key display component for visually representing keyboard shortcuts, hotkeys, and key combinations. Renders with a key-cap style appearance including borders, shadows, and subtle depth to mimic physical keyboard keys.

## Installation & Integration

### Import

```tsx
import { Kbd } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop      | Type                                                                     | Default     | Description               |
| --------- | ------------------------------------------------------------------------ | ----------- | ------------------------- |
| size      | `xs \| sm \| base \| lg`                                                | `sm`        | Key text size variant     |
| variant   | `default \| primary \| secondary \| success \| warning \| error \| info` | `default`   | Key color variant         |
| className | `string`                                                                 | -           | Additional CSS classes    |

Inherits all standard HTML `kbd` element attributes.

## Accessibility

- Uses semantic `<kbd>` element for proper keyboard key identification
- Screen readers announce the key text content naturally
- Minimum width ensures touch-friendly sizing on mobile devices
- Color variants maintain readable contrast for WCAG 2.1 Level AA
- Key-cap visual styling provides clear distinction from surrounding text

## Component API

| Export   | Type                              | Description                        |
