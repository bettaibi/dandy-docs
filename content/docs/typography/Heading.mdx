# Heading

A semantic heading component that renders `h1` through `h6` elements with independent size control. The `level` prop determines the HTML element while `size` allows visual flexibility, enabling proper document structure without being locked to default heading sizes.

## Installation & Integration

### Import

```tsx
import { Heading } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop        | Type                                                        | Default     | Description                                         |
| ----------- | ----------------------------------------------------------- | ----------- | --------------------------------------------------- |
| level       | `1 \| 2 \| 3 \| 4 \| 5 \| 6`                              | `1`         | Heading level, determines the HTML element           |
| size        | `xs \| sm \| base \| lg \| xl \| 2xl \| 3xl \| 4xl`       | -           | Size variant (defaults based on level if not set)    |
| weight      | `normal \| medium \| semibold \| bold`                      | `semibold`  | Text font weight                                    |
| variant     | `default \| muted \| success \| warning \| error \| info`  | `default`   | Text color variant                                  |
| lineThrough | `boolean`                                                   | `false`     | Apply line-through text decoration                  |
| uppercase   | `boolean`                                                   | `false`     | Transform text to uppercase                         |
| underline   | `boolean`                                                   | `false`     | Apply underline text decoration                     |
| truncate    | `boolean`                                                   | `false`     | Truncate text with ellipsis on overflow              |
| trim        | `boolean`                                                   | `false`     | Trim leading and trailing whitespace                 |
| wrap        | `boolean`                                                   | `false`     | Prevent text wrapping                               |
| className   | `string`                                                    | -           | Additional CSS classes                               |

**Default size mapping:** h1=4xl, h2=3xl, h3=2xl, h4=xl, h5=lg, h6=base.

Inherits all standard HTML heading element attributes.

## Accessibility

- Renders semantic heading elements (`h1`-`h6`) for proper document outline
- Heading levels should follow a logical hierarchy (do not skip levels)
- Color variants maintain sufficient contrast ratios for WCAG 2.1 Level AA
- Includes `tracking-tight` for improved readability at large sizes
- Screen readers announce the correct heading level for navigation
