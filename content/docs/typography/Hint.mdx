# Hint

A lightweight helper text component for displaying captions, descriptions, and supplementary content. Renders as a muted `span` element with compact sizing, ideal for form hints, image captions, and secondary labels.

## Installation & Integration

### Import

```tsx
import { Hint } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop      | Type        | Default | Description            |
| --------- | ----------- | ------- | ---------------------- |
| size      | `xs \| sm`  | `sm`    | Hint text size variant |
| className | `string`    | -       | Additional CSS classes |

Inherits all standard HTML `span` element attributes.

## Accessibility

- Renders as a semantic `span` element
- Uses muted foreground color for visual hierarchy without hiding content
- Text remains readable at both `xs` and `sm` sizes across viewports
- Can be associated with form controls via `id` and `aria-describedby`
