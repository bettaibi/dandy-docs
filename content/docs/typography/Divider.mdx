# Divider

A visual separator component for creating clear boundaries between content sections. Supports horizontal and vertical orientations, multiple visual variants, configurable thickness, responsive spacing, and optional center content such as text or icons.

## Installation & Integration

### Import

```tsx
import { Divider } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop        | Type                           | Default        | Description                                     |
| ----------- | ------------------------------ | -------------- | ----------------------------------------------- |
| variant     | `default \| trimmed \| dot`    | `default`      | Visual style of the divider                     |
| orientation | `horizontal \| vertical`       | `horizontal`   | Direction of the divider                        |
| size        | `sm \| md \| lg`               | `sm`           | Thickness of the divider line                   |
| spacing     | `sm \| md \| lg \| xl \| none` | `none`         | Responsive spacing around the divider           |
| children    | `React.ReactNode`              | -              | Optional text or icon to display in the center  |
| className   | `string`                       | -              | Additional CSS classes                          |

Inherits all standard HTML `hr` element attributes.

## Accessibility

- Uses semantic `<hr>` element with implicit `role="separator"`
- When content is present, it remains properly exposed to screen readers
- Theme colors ensure sufficient contrast in both light and dark modes
- Vertical dividers provide visual separation without disrupting document flow
