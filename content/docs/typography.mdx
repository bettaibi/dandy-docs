# Typography

Typography in Dandy UI is designed to be both elegant and readable, providing a solid foundation for your application's text content.

## Text Component

The `Text` component is the primary way to display text. It offers various props to control the appearance and behavior of the text.

<Tabs items={["Preview", "Props"]}>
  <Tab value="Preview">
    <p>This is a paragraph of text.</p>
  </Tab>
  <Tab value="Props">
    <TypeTable
      type={{
        as: {
          description: "The HTML element to render the text as.",
          type: "enum",
          entries: ["p", "span", "div", "label"],
          default: "p",
        },
        className: {
          description: "Additional CSS classes to apply to the text.",
          type: "string",
        },
        size: {
          description: "The size of the text.",
          type: "enum",
          entries: [
            "xs",
            "sm",
            "md",
            "lg",
            "xl",
            "2xl",
            "3xl",
            "4xl",
            "5xl",
            "6xl",
          ],
        },
        weight: {
          description: "The font weight of the text.",
          type: "enum",
          entries: ["normal", "medium", "semibold", "bold"],
        },
        italic: {
          description: "Whether the text should be italicized.",
          type: "boolean",
        },
        underline: {
          description: "Whether the text should be underlined.",
          type: "boolean",
        },
      }}
    />
  </Tab>
</Tabs>
