# Popover

A floating popover panel built on Radix UI Popover. Anchors to a trigger element with automatic collision detection, configurable side and alignment, and smooth animations.

## Installation & Integration

### Import

```tsx
import { Popover } from "dandy-ui";
```

### Peer Dependencies

```bash
npm install @radix-ui/react-popover
```

## Props

### Popover.Content

| Prop             | Type                                               | Default    | Description                      |
| ---------------- | -------------------------------------------------- | ---------- | -------------------------------- |
| side             | `"top" \| "right" \| "bottom" \| "left"`           | `"bottom"` | Preferred side of the trigger    |
| align            | `"start" \| "center" \| "end"`                     | `"center"` | Alignment against the trigger    |
| sideOffset       | `number`                                           | `8`        | Distance in px from the trigger  |
| collisionPadding | `number`                                           | `12`       | Padding in px from viewport edge |
| rounded          | `"sm" \| "md" \| "lg" \| "xl" \| "full" \| "none"` | -          | Border radius override           |
| className        | `string`                                           | -          | Additional CSS classes           |

Inherits all Radix `Popover.Content` attributes.

## Accessibility

- Built on Radix UI Popover for full WCAG compliance
- `Escape` key closes the popover
- Focus returns to trigger on close
- Smart collision detection prevents off-screen rendering
- Respects `prefers-reduced-motion`
- Screen reader friendly with proper ARIA attributes

## Component API

| Export              | Type                              | Description                    |
| ------------------- | --------------------------------- | ------------------------------ |
| Popover             | Compound component                | Root popover controller        |
| Popover.Trigger     | `Radix PopoverTrigger`            | Element that opens the popover |
| Popover.Content     | `React.ForwardRefExoticComponent` | Popover content container      |
| PopoverContentProps | `interface`                       | Props for Popover.Content      |
