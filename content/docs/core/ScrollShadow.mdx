# ScrollShadow

A CSS-only scroll container that applies gradient shadows at the edges to visually indicate scrollable content. Supports vertical and horizontal scrolling with configurable shadow intensity. Works in both light and dark modes.

## Installation & Integration

### Import

```tsx
import { ScrollShadow } from 'dandy-ui';
import 'dandy-ui/theme.css';
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| direction | `"horizontal" \| "vertical"` | `"vertical"` | Scroll direction and shadow placement |
| elevation | `"sm" \| "md" \| "lg"` | `"md"` | Shadow size and intensity |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `div` attributes (`style`, `id`, etc.).

## Accessibility

- Shadow overlays use `aria-hidden="true"` so they are invisible to screen readers
- Uses `pointer-events-none` on shadows to avoid blocking scroll interaction
- Scrollable content area maintains native scroll behavior and keyboard navigation
- Supports all standard ARIA attributes via prop spread

## Component API

| Export | Type | Description |
