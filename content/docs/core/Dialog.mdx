# Dialog

A modal dialog built on Radix UI Dialog primitive. Provides focus trapping, keyboard dismissal, scroll locking, and configurable animations. Supports multiple sizes, placements, and fullscreen modes.

## Installation & Integration

### Import

```tsx
import { Dialog } from "dandy-ui";
```

### Peer Dependencies

```bash
npm install @radix-ui/react-dialog
```

## Props

### Dialog.Content

| Prop         | Type                                             | Default      | Description                                |
| ------------ | ------------------------------------------------ | ------------ | ------------------------------------------ |
| size         | `"auto" \| "xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`       | Dialog width                               |
| backdropExit | `boolean`                                        | `true`       | Whether clicking outside closes the dialog |
| overlay      | `"blur" \| "transparent" \| "none"`              | `"blur"`     | Backdrop overlay style                     |
| placement    | `"centered" \| "top"`                            | `"centered"` | Dialog position                            |
| animate      | `"fadeIn" \| "slideIn"`                          | `"fadeIn"`   | Animation type                             |
| fullscreen   | `"mobile-only" \| "desktop-only" \| "always"`    | -            | Fullscreen behavior                        |
| rounded      | `"sm" \| "md" \| "lg" \| "xl"`                   | -            | Border radius override                     |
| className    | `string`                                         | -            | Additional CSS classes                     |

Inherits all Radix `Dialog.Content` attributes.

## Accessibility

- Built on Radix UI Dialog for full WCAG compliance
- Focus trap when open, focus returns to trigger on close
- `Escape` key closes the dialog
- Background scroll is prevented while open
- `aria-labelledby` linked to `Dialog.Title`
- `aria-describedby` linked to `Dialog.Description`
- Respects `prefers-reduced-motion`

## Component API

| Export             | Type                              | Description                    |
| ------------------ | --------------------------------- | ------------------------------ |
| Dialog             | Compound component                | Root dialog controller         |
| Dialog.Trigger     | `Radix DialogTrigger`             | Element that opens the dialog  |
| Dialog.Content     | `React.ForwardRefExoticComponent` | Dialog content container       |
| Dialog.Close       | `Radix DialogClose`               | Element that closes the dialog |
| Dialog.Overlay     | `React.ForwardRefExoticComponent` | Backdrop overlay               |
| Dialog.Header      | `React.ForwardRefExoticComponent` | Header layout container        |
| Dialog.Footer      | `React.ForwardRefExoticComponent` | Footer layout container        |
| Dialog.Title       | `React.ForwardRefExoticComponent` | Dialog title heading           |
| Dialog.Description | `React.ForwardRefExoticComponent` | Dialog description text        |
| DialogContentProps | `interface`                       | Props for Dialog.Content       |
| DialogHeaderProps  | `type`                            | Props for Dialog.Header        |
| DialogFooterProps  | `type`                            | Props for Dialog.Footer        |
