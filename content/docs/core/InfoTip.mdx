# InfoTip

A tooltip component built on Radix UI Tooltip with support for multiple theme-aware color variants, responsive sizing, and configurable placement. Shows contextual information on hover or focus.

## Installation & Integration

### Import

```tsx
import { InfoTip } from "dandy-ui";
```

### Peer Dependencies

```bash
npm install @radix-ui/react-tooltip
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | `React.ReactElement` | required | The trigger element |
| content | `React.ReactNode` | required | Content to display in the tooltip |
| variant | `"default" \| "primary" \| "secondary" \| "accent" \| "success" \| "warning" \| "error" \| "info"` | `"default"` | Color variant |
| size | `"sm" \| "md" \| "lg"` | `"md"` | Tooltip size |
| placement | `"top" \| "bottom" \| "left" \| "right"` | `"top"` | Position relative to trigger |
| hidden | `boolean` | `false` | Prevents tooltip from showing |
| delayDuration | `number` | `0` | Delay before showing (ms) |
| rounded | `"sm" \| "md" \| "lg" \| "xl" \| "full" \| "none"` | - | Border radius override |
| className | `string` | - | Additional CSS classes for tooltip content |

## Accessibility

- Built on Radix UI Tooltip for full WCAG compliance
- Appears on both hover and focus for keyboard users
- Respects `prefers-reduced-motion`
- Responsive max-width adapts across breakpoints
- Arrow indicator matches variant background color
