# Select

A customizable dropdown select component built on Radix UI primitives. Features semantic color variants, flexible sizing, keyboard navigation, and mobile-first design.

## Installation & Integration

### Import

```tsx
import {
  Select,
  SelectTrigger,
  SelectContent,
  SelectValue,
  SelectItem,
  SelectGroup,
  SelectLabel,
  SelectSeparator,
} from "dandy-ui";
```

### Peer Dependencies

This component requires the following peer dependency:

```bash
npm install @radix-ui/react-select
```

## Props

### Select

| Prop          | Type                                             | Default   | Description                              |
| ------------- | ------------------------------------------------ | --------- | ---------------------------------------- |
| variant       | `default \| success \| error \| warning \| info` | `default` | Visual style variant for focus and items |
| value         | `string`                                         | -         | Controlled selected value                |
| defaultValue  | `string`                                         | -         | Default selected value                   |
| onValueChange | `(value: string) => void`                        | -         | Callback when selection changes          |
| disabled      | `boolean`                                        | `false`   | Disables the select                      |
| required      | `boolean`                                        | `false`   | Makes the select required                |
| name          | `string`                                         | -         | Name for form submission                 |
| open          | `boolean`                                        | -         | Controlled open state                    |
| defaultOpen   | `boolean`                                        | -         | Default open state                       |
| onOpenChange  | `(open: boolean) => void`                        | -         | Callback when open state changes         |

### SelectTrigger

| Prop      | Type                                             | Default   | Description                   |
| --------- | ------------------------------------------------ | --------- | ----------------------------- |
| variant   | `default \| success \| error \| warning \| info` | inherited | Visual style variant override |
| size      | `sm \| md \| lg`                                 | `md`      | Trigger size                  |
| leftSlot  | `React.ReactNode`                                | -         | Content in the left slot      |
| rounded   | `sm \| md \| lg \| xl`                           | -         | Corner radius override        |
| className | `string`                                         | -         | Additional CSS classes        |

Inherits all standard HTML button attributes.

### SelectContent

| Prop       | Type                     | Default  | Description                      |
| ---------- | ------------------------ | -------- | -------------------------------- |
| position   | `item-aligned \| popper` | `popper` | Position preference for dropdown |
| rounded    | `sm \| md \| lg \| xl`   | -        | Corner radius override           |
| sideOffset | `number`                 | `4`      | Distance from trigger            |
| className  | `string`                 | -        | Additional CSS classes           |

### SelectItem

| Prop      | Type                                                       | Default   | Description                   |
| --------- | ---------------------------------------------------------- | --------- | ----------------------------- |
| value     | `string`                                                   | -         | Option value (required)       |
| variant   | `default \| success \| error \| warning \| info \| accent` | inherited | Visual style variant override |
| disabled  | `boolean`                                                  | `false`   | Disables the item             |
| className | `string`                                                   | -         | Additional CSS classes        |

### SelectLabel

| Prop      | Type     | Default | Description            |
| --------- | -------- | ------- | ---------------------- |
| className | `string` | -       | Additional CSS classes |

### SelectGroup

Standard Radix Select Group component with no custom props.

### SelectSeparator

| Prop      | Type     | Default | Description            |
| --------- | -------- | ------- | ---------------------- |
| className | `string` | -       | Additional CSS classes |

### SelectValue

| Prop        | Type     | Default | Description                 |
| ----------- | -------- | ------- | --------------------------- |
| placeholder | `string` | -       | Placeholder text when empty |

## Accessibility

- Keyboard navigable with Tab, Space/Enter to open, Arrow keys to navigate
- Type-ahead search to quickly find options
- Escape to close the dropdown
- Focus indicators on trigger and items
- Proper ARIA attributes for screen readers
- Supports disabled state with proper communication
- Minimum 44x44px touch targets for mobile
- WCAG 2.1 Level AA compliant
- Respects prefers-reduced-motion for animations

## Component API

| Export                 | Type                              | Description                            |
| ---------------------- | --------------------------------- | -------------------------------------- |
| Select                 | `React.FC`                        | Root component with variant context    |
| SelectTrigger          | `React.ForwardRefExoticComponent` | Trigger button with ref forwarding     |
| SelectContent          | `React.ForwardRefExoticComponent` | Dropdown content container             |
| SelectValue            | `React.ForwardRefExoticComponent` | Displays selected value or placeholder |
| SelectItem             | `React.ForwardRefExoticComponent` | Individual selectable option           |
| SelectGroup            | `React.ForwardRefExoticComponent` | Groups related items                   |
| SelectLabel            | `React.ForwardRefExoticComponent` | Labels a group of items                |
| SelectSeparator        | `React.ForwardRefExoticComponent` | Visual separator between groups        |
| SelectScrollUpButton   | `React.ForwardRefExoticComponent` | Scroll indicator at top                |
| SelectScrollDownButton | `React.ForwardRefExoticComponent` | Scroll indicator at bottom             |
| SelectProps            | `interface`                       | Select component props                 |
| SelectTriggerProps     | `interface`                       | SelectTrigger component props          |
| SelectContentProps     | `interface`                       | SelectContent component props          |
| SelectItemProps        | `interface`                       | SelectItem component props             |
