# CheckBox

A customizable checkbox component with native browser functionality, theme-aware styling, and multiple visual variants. Built on the native HTML checkbox input for full keyboard support and accessibility.

## Installation & Integration

### Import

```tsx
import { CheckBox } from "dandy-ui";
```

## Props

| Prop         | Type                                             | Default   | Description                        |
| ------------ | ------------------------------------------------ | --------- | ---------------------------------- |
| variant      | `default \| success \| error \| warning \| info` | `default` | Visual style variant               |
| size         | `sm \| md \| lg`                                 | `md`      | Checkbox dimensions                |
| rounded      | `sm \| md \| lg \| none`                         | `md`      | Border radius size                 |
| errorMessage | `string`                                         | -         | Error message displayed in tooltip |
| className    | `string`                                         | -         | Additional CSS classes             |

Inherits all standard HTML input attributes (`onChange`, `checked`, `defaultChecked`, `disabled`, `name`, `value`, etc.).

## Accessibility

- Native checkbox input for full keyboard support
- Keyboard navigable with Tab and Space keys
- Clear focus indicators with colored ring glow
- WCAG 2.1 Level AA compliant with sufficient color contrast
- Screen reader compatible with proper semantic elements
- Respects `prefers-reduced-motion` for smooth transitions
- Error messages announced via InfoTip tooltip
