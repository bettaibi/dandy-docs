# InputOTP

Multi-slot input component for one-time passwords, verification codes, and PIN entry with built-in validation states and error handling.

## Installation & Integration

### Import

```tsx
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSlot,
  InputOTPSeparator,
} from "dandy-ui";
import "dandy-ui/theme.css";
```

### Peer Dependencies

This component requires the following peer dependency:

```bash
npm install input-otp
```

## Props

### InputOTP

| Prop               | Type                                             | Default   | Description                  |
| ------------------ | ------------------------------------------------ | --------- | ---------------------------- |
| variant            | `default \| success \| error \| warning \| info` | `default` | Visual style variant         |
| otpSize            | `sm \| md \| lg`                                 | `md`      | Slot dimensions              |
| errorMessage       | `string`                                         | `""`      | Error message to display     |
| containerClassName | `string`                                         | -         | Container wrapper class      |
| rounded            | `sm \| md \| lg \| xl`                           | -         | Border radius override       |
| maxLength          | `number`                                         | -         | Maximum number of characters |
| value              | `string`                                         | -         | Controlled value             |
| onChange           | `(value: string) => void`                        | -         | Value change handler         |
| disabled           | `boolean`                                        | `false`   | Disables input               |
| className          | `string`                                         | -         | Additional CSS classes       |

Inherits all standard `input-otp` OTPInput attributes (`textAlign`, `pattern`, `pushPasswordManagerStrategy`, etc.).

### InputOTPSlot

| Prop      | Type                                             | Default | Description              |
| --------- | ------------------------------------------------ | ------- | ------------------------ |
| index     | `number`                                         | -       | Slot position (required) |
| variant   | `default \| success \| error \| warning \| info` | -       | Override variant         |
| slotSize  | `sm \| md \| lg`                                 | -       | Override size            |
| rounded   | `sm \| md \| lg \| xl`                           | -       | Override border radius   |
| className | `string`                                         | -       | Additional CSS classes   |

Inherits all standard HTML `div` attributes.

### InputOTPGroup

| Prop      | Type     | Default | Description            |
| --------- | -------- | ------- | ---------------------- |
| className | `string` | -       | Additional CSS classes |

Inherits all standard HTML `div` attributes.

### InputOTPSeparator

| Prop          | Type             | Default | Description             |
| ------------- | ---------------- | ------- | ----------------------- |
| separatorSize | `sm \| md \| lg` | -       | Override separator size |
| className     | `string`         | -       | Additional CSS classes  |

Inherits all standard HTML `div` attributes.

## Accessibility

- Full keyboard navigation support with native input behavior
- Focus indicators with ring styling on active slots
- Screen reader friendly with proper semantic elements
- Clear visual feedback for active, filled, and error states
- WCAG 2.1 Level AA compliant with high contrast ratios
- Minimum 44px touch targets on large size for mobile usability
- Animated caret indicator for clear input position feedback

## Component API

| Export                 | Type                              | Description                       |
| ---------------------- | --------------------------------- | --------------------------------- |
| InputOTP               | `React.ForwardRefExoticComponent` | Main OTP input component with ref |
| InputOTPProps          | `interface`                       | InputOTP component props          |
| InputOTPGroup          | `React.ForwardRefExoticComponent` | Group container for OTP slots     |
| InputOTPGroupProps     | `interface`                       | InputOTPGroup component props     |
| InputOTPSlot           | `React.ForwardRefExoticComponent` | Individual OTP character slot     |
| InputOTPSlotProps      | `interface`                       | InputOTPSlot component props      |
| InputOTPSeparator      | `React.ForwardRefExoticComponent` | Visual separator between groups   |
| InputOTPSeparatorProps | `interface`                       | InputOTPSeparator component props |
