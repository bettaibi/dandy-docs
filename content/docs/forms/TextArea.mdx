# TextArea

Multi-line text input component for collecting longer text content. Features semantic color variants that appear on focus, flexible sizing, resize controls, and built-in error validation with tooltips.

## Installation & Integration

### Import

```tsx
import { TextArea } from "dandy-ui";
```

## Props

| Prop         | Type                                             | Default    | Description                    |
| ------------ | ------------------------------------------------ | ---------- | ------------------------------ |
| variant      | `default \| success \| error \| warning \| info` | `default`  | Visual style variant           |
| size         | `sm \| md \| lg`                                 | `md`       | Component size                 |
| resize       | `none \| vertical \| horizontal \| both`         | `vertical` | Resize behavior                |
| errorMessage | `string`                                         | `""`       | Error message shown in tooltip |
| rounded      | `sm \| md \| lg \| xl \| full \| none`           | -          | Border radius size             |
| className    | `string`                                         | -          | Additional CSS classes         |

Inherits all standard HTML textarea attributes (`rows`, `cols`, `maxLength`, `onChange`, etc.).

## Accessibility

- Focus visible with colored ring indicator matching variant
- Error states announced via tooltip (InfoTip component)
- Supports all standard textarea ARIA attributes
- Keyboard navigable with Tab key
- WCAG 2.1 Level AA compliant
- Respects user's motion preferences with reduced animations
- Proper disabled and read-only state handling
