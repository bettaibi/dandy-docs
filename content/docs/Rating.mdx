# Rating

A flexible rating component for displaying and collecting user ratings. Supports both read-only displays and interactive rating selection with optional half-star precision.

## Installation & Integration

### Import

```tsx
import { Rating } from "dandy-ui";
```

## Props

| Prop          | Type                                                                               | Default  | Description                                              |
| ------------- | ---------------------------------------------------------------------------------- | -------- | -------------------------------------------------------- |
| value         | `number`                                                                           | `0`      | Current rating value (0 to max). Supports decimal values |
| max           | `number`                                                                           | `5`      | Maximum number of stars to display                       |
| size          | `sm \| md \| lg`                                                                   | `md`     | Star size (16px / 20px / 24px)                           |
| interactive   | `boolean`                                                                          | `false`  | Enable interactive rating selection                      |
| onValueChange | `(value: number) => void`                                                          | -        | Callback when rating changes (interactive mode only)     |
| showValue     | `boolean`                                                                          | `false`  | Display numeric rating value                             |
| variant       | `default \| accent \| warning \| primary \| secondary \| success \| error \| info` | `accent` | Color variant for filled stars                           |
| allowHalf     | `boolean`                                                                          | `false`  | Enable half-star ratings                                 |
| ariaLabel     | `string`                                                                           | -        | Accessible label for screen readers                      |
| className     | `string`                                                                           | -        | Additional CSS classes                                   |

Inherits all standard HTML `div` attributes.

## Accessibility

- Keyboard navigable with Tab key (interactive mode)
- Enter/Space keys to select rating (interactive mode)
- Focus visible with ring indicator
- Screen reader announces current rating
- Interactive stars have role="button" with descriptive labels
- Non-interactive ratings use role="img" with aria-label
- WCAG 2.1 Level AA compliant
