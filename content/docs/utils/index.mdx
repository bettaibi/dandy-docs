# Utilities

A collection of tree-shakeable utility functions built on native `Intl` APIs. Zero external dependencies -- just import what you need.

```tsx
import { formatDate, formatCurrency, debounce } from 'dandy-ui/utils';
```

---

| Group | Description |
| ----- | ----------- |
| [Date Utilities](/docs/utils/DateUtilities) | Format, compare, and manipulate dates with `Intl.DateTimeFormat` |
| [Number Utilities](/docs/utils/NumberUtilities) | Format numbers, currencies, percentages, and lists |
| [Input / Display Utilities](/docs/utils/InputDisplayUtilities) | Phone numbers, credit cards, file sizes, and HTML sanitization |
| [Validation Utilities](/docs/utils/ValidationUtilities) | Validate emails, phone numbers, and URLs |
| [Async Utilities](/docs/utils/AsyncUtilities) | Debounce, throttle, and clipboard helpers |
| [Hooks](/docs/utils/Hooks) | React hooks for media queries and theme management |
