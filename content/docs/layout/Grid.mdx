# Grid

A CSS grid container with responsive column and row configuration. Supports breakpoint-aware layouts with a companion `GridItem` for controlling span behavior.

## Installation & Integration

### Import

```tsx
import { Grid, GridItem } from "dandy-ui";
```

## Props

### Grid

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| cols | `ColCount \| { base?: ColCount; sm?: ColCount; md?: ColCount; lg?: ColCount; xl?: ColCount }` | `1` | Number of columns (1-12), supports responsive object |
| rows | `RowCount \| { base?: RowCount; sm?: RowCount; md?: RowCount; lg?: RowCount; xl?: RowCount }` | - | Number of rows (1-6), supports responsive object |
| gap | `"none" \| "xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | Gap between grid items |
| as | `React.ElementType` | `"div"` | Render as a different HTML element |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `div` attributes.

### GridItem

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| span | `SpanValue \| { base?: SpanValue; sm?: SpanValue; md?: SpanValue; lg?: SpanValue; xl?: SpanValue }` | - | Columns to span (1-12 or "full"), supports responsive object |
| rowSpan | `RowCount \| { base?: RowCount; sm?: RowCount; md?: RowCount; lg?: RowCount; xl?: RowCount }` | - | Rows to span (1-6), supports responsive object |
| as | `React.ElementType` | `"div"` | Render as a different HTML element |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `div` attributes.

## Accessibility

- Renders as a semantic `div` by default, configurable via `as` prop
- Grid layout is transparent to screen readers; content order matches DOM order
- Supports all standard ARIA attributes via prop spread
