# Container

A responsive wrapper that constrains content to a maximum width with consistent horizontal padding. Automatically centers content and adapts padding across breakpoints.

## Installation & Integration

### Import

```tsx
import { Container } from "dandy-ui";
```

## Non-Centered

By default, the container is centered with `mx-auto`. Set `centered={false}` to disable this.

  </Tab>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| maxWidth | `"sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "4xl" \| "5xl" \| "6xl" \| "7xl" \| "full"` | `"lg"` | Maximum width of the container |
| centered | `boolean` | `true` | Whether to center the container with auto margins |
| hidden | `"sm" \| "md" \| "lg" \| "xl"` | - | Hide content at the specified breakpoint and above |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `div` attributes (except `hidden`).

## Accessibility

- Uses semantic `div` element for generic content grouping
- Responsive padding scales from `px-4` on mobile to `px-8` on large screens
- Supports all standard ARIA attributes via prop spread
