# Paper

A theme-aware surface container that groups related content. Supports outlined borders, shadow elevation levels, and configurable rounded corners for visual hierarchy.

## Installation & Integration

### Import

```tsx
import { Paper } from "dandy-ui";
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| outlined | `boolean` | `false` | Whether to show a border |
| elevation | `"none" \| "sm" \| "md" \| "lg" \| "xl"` | `"none"` | Shadow elevation level |
| rounded | `"none" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "3xl" \| "full"` | - | Border radius override |
| hidden | `"sm" \| "md" \| "lg" \| "xl"` | - | Hide content at the specified breakpoint and above |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `div` attributes (except `hidden`).

## Accessibility

- Uses theme-aware `bg-card` and `text-card-foreground` for proper contrast
- Works with dark mode out of the box
- Supports all standard ARIA attributes via prop spread
- Use `role` or semantic children for meaningful landmark structure
