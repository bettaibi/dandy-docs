# Table

A mobile-first, compound data table component for displaying tabular data. Supports theme-aware variants, row selection, sticky headers, striped rows, and responsive column visibility.

## Installation & Integration

### Import

```tsx
import { Table } from "dandy-ui";
```

## Props

### Table

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "error" \| "info"` | `"default"` | Theme-aware color variant for hover and selection states |
| size | `"sm" \| "md" \| "lg"` | `"md"` | Controls row padding and text size |
| striped | `boolean` | `false` | Applies alternating row background colors |
| outlined | `boolean` | `false` | Adds borders to the table and cells |
| sticky | `boolean` | `false` | Makes the table header sticky on scroll |
| rounded | `"none" \| "sm" \| "md" \| "lg" \| "xl"` | - | Border radius of the table container |
| elevation | `"none" \| "xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"sm"` | Shadow elevation of the table container |
| selectable | `"single" \| "multiple"` | - | Enables row selection mode |
| selectedKeys | `string[]` | - | Array of selected row keys (controlled) |
| onSelectionChange | `(keys: string[]) => void` | - | Callback fired when selection changes |
| emptyState | `React.ReactNode` | - | Content displayed when table body is empty |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `table` attributes.

### Table.Row

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| rowKey | `string` | - | Unique identifier for selection tracking |
| isSelected | `boolean` | - | Controlled selected state |
| disableRowSelect | `boolean` | `false` | Prevents row from being selected |
| onRowClick | `(event: React.MouseEvent) => void` | - | Callback fired when row is clicked |
| className | `string` | - | Additional CSS classes |

### Table.Column

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| hiddenAt | `"xs" \| "sm" \| "md" \| "lg"` | - | Breakpoint at which column becomes hidden |
| width | `string \| number` | - | Column width |
| align | `"left" \| "center" \| "right"` | `"left"` | Text alignment |
| className | `string` | - | Additional CSS classes |

### Table.Cell

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| hiddenAt | `"xs" \| "sm" \| "md" \| "lg"` | - | Breakpoint at which cell becomes hidden |
| width | `string \| number` | - | Cell width |
| align | `"left" \| "center" \| "right"` | `"left"` | Text alignment |
| className | `string` | - | Additional CSS classes |

### Table.Head / Table.Body / Table.Footer

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | `string` | - | Additional CSS classes |
| children | `React.ReactNode` | - | Row content |

## Accessibility

- Renders as native `<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<tr>`, `<th>`, `<td>` elements
- `Table.Column` uses `scope="col"` for proper header association
- Table container uses `role="region"` with `aria-label="Data table"`
- Row selection supports keyboard navigation with Enter and Space keys
- Selected rows use `aria-selected` attribute
- Disabled rows use `aria-disabled` attribute
- Select-all checkbox in `Table.Head` supports indeterminate state
- Focus ring indicator for keyboard-navigable rows
