## Props

| Prop      | Type                                                                               | Default        | Description                     |
| --------- | ---------------------------------------------------------------------------------- | -------------- | ------------------------------- |
| size      | `xs \| sm \| md \| lg \| xl \| 2xl \| 3xl`                                         | `md`           | Spinner size                    |
| variant   | `default \| primary \| secondary \| accent \| success \| warning \| error \| info` | `default`      | Color variant                   |
| label     | `string`                                                                           | `"Loading..."` | Loading text for screen readers |
| className | `string`                                                                           | -              | Additional CSS classes          |

Inherits all standard HTML `div` attributes (`role`, `aria-*`, etc.).

## Accessibility

- Uses `role="status"` for live region announcements
- Includes screen reader text via `label` prop (visually hidden)
- Icon marked with `aria-hidden="true"` to avoid duplication
- Respects user's motion preferences (animation can be disabled via CSS)
- Semantic HTML structure for assistive technologies
