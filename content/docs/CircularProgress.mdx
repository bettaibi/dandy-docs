# CircularProgress

A mobile-first circular countdown progress indicator that automatically decrements every second. Features animated circular progress bar synchronized with countdown, multiple size and variant options, and theme-aware design with full dark mode support.

## Installation & Integration

### Import

```tsx
import { CircularProgress } from "dandy-ui";
import "dandy-ui/theme.css";
```

## Props

| Prop        | Type                                                                                               | Default     | Description                                                       |
| ----------- | -------------------------------------------------------------------------------------------------- | ----------- | ----------------------------------------------------------------- |
| value       | `number`                                                                                           | `60`        | Initial countdown value                                           |
| size        | `"sm" \| "md" \| "lg" \| "xl"`                                                                     | `"md"`      | Size of the circular progress                                     |
| variant     | `"default" \| "primary" \| "secondary" \| "accent" \| "success" \| "warning" \| "error" \| "info"` | `"default"` | Visual style variant                                              |
| strokeWidth | `number`                                                                                           | -           | Stroke width of progress circle (auto-calculated if not provided) |
| showValue   | `boolean`                                                                                          | `true`      | Whether to show the countdown number                              |
| onComplete  | `() => void`                                                                                       | -           | Callback when countdown reaches zero                              |
| active      | `boolean`                                                                                          | `true`      | Whether the countdown is active/running                           |
| suffix      | `string`                                                                                           | -           | Custom label suffix (e.g., "s", "%", "min")                       |
| className   | `string`                                                                                           | -           | Additional CSS classes                                            |

Inherits all standard HTML `div` attributes.

## Accessibility

- Uses `role="timer"` for proper semantics
- Includes `aria-live="polite"` for screen reader announcements
- Provides descriptive `aria-label` with current countdown value
- Includes `aria-valuemin`, `aria-valuemax`, and `aria-valuenow` for progress tracking
- SVG elements marked with `aria-hidden="true"` to avoid redundant announcements
- Supports `prefers-reduced-motion` with conditional transitions
- Tabular numbers for consistent digit spacing
