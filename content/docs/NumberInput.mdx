# NumberInput

A number input component with built-in increment and decrement controls, validation states, and customizable placement options.

## Installation & Integration

### Import

```tsx
import { NumberInput } from "dandy-ui";
```

## Props

| Prop              | Type                                             | Default   | Description                                 |
| ----------------- | ------------------------------------------------ | --------- | ------------------------------------------- |
| variant           | `default \| success \| error \| warning \| info` | `default` | Visual style variant with validation colors |
| size              | `sm \| md \| lg`                                 | `md`      | Component size                              |
| controlsPlacement | `start \| end \| none`                           | `end`     | Placement of increment/decrement controls   |
| step              | `number`                                         | `1`       | Step value for increment/decrement          |
| min               | `number`                                         | -         | Minimum value                               |
| max               | `number`                                         | -         | Maximum value                               |
| value             | `number`                                         | -         | Current value                               |
| handleValueChange | `(value: number) => void`                        | -         | Callback when value changes via controls    |
| errorMessage      | `string`                                         | `""`      | Error message to display in tooltip         |
| leftSlot          | `React.ReactNode`                                | -         | Content to render on the left side          |
| rightSlot         | `React.ReactNode`                                | -         | Content to render on the right side         |
| rounded           | `sm \| md \| lg \| xl \| full \| none`           | -         | Border radius size                          |
| disabled          | `boolean`                                        | `false`   | Disables the input and controls             |
| className         | `string`                                         | -         | Additional CSS classes                      |

Inherits all standard HTML input attributes (`placeholder`, `onChange`, `onBlur`, etc.).

## Accessibility

- Keyboard navigable with Tab key
- Increment/decrement controls are hidden from tab order (tabIndex={-1})
- Focus visible with ring indicator matching variant color
- Native number input spinners are hidden for consistent styling
- Error states automatically shown via InfoTip tooltip
- Buttons are automatically disabled when min/max limits are reached
- WCAG 2.1 Level AA compliant
- Minimum 32px height on small size, 40px on medium, 48px on large for touch targets
- Screen reader compatible with proper semantic elements
