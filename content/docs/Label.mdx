## Props

| Prop      | Type                                                        | Default   | Description                              |
| --------- | ----------------------------------------------------------- | --------- | ---------------------------------------- |
| variant   | `default \| success \| error \| warning \| info \| primary` | `default` | Semantic color variant                   |
| size      | `sm \| md \| lg`                                            | `md`      | Label size                               |
| required  | `boolean`                                                   | `false`   | Shows required indicator (asterisk)      |
| disabled  | `boolean`                                                   | `false`   | Applies disabled styling                 |
| htmlFor   | `string`                                                    | -         | Associates label with form control by ID |
| className | `string`                                                    | -         | Additional CSS classes                   |

Inherits all standard HTML label attributes.

## Accessibility

- Native `<label>` element for proper form association
- Use `htmlFor` prop to associate with form controls by ID
- Required indicator announced by screen readers with `aria-label="required"`
- Sufficient color contrast in all variants (WCAG 2.1 Level AA)
- Disabled state uses reduced opacity and default cursor
- Screen reader compatible with semantic HTML
- Smooth transitions respect user's motion preferences
