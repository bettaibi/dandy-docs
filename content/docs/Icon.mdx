# Icon

A flexible, reusable icon wrapper that ensures consistency across the application. Works with any icon implementation: SVGs, Material Icons, FontAwesome, or custom icons. Provides standardized sizes, theme-aware color variants, and consistent spacing.

## Installation & Integration

### Import

```tsx
import { Icon } from "dandy-ui";
```

## Props

| Prop      | Type                                                                                        | Default   | Description                                           |
| --------- | ------------------------------------------------------------------------------------------- | --------- | ----------------------------------------------------- |
| value     | `React.ReactNode`                                                                           | -         | Icon content (SVG, Material Icons, FontAwesome, etc.) |
| size      | `xs \| sm \| md \| lg \| xl \| 2xl \| 3xl`                                                  | `md`      | Icon size                                             |
| variant   | `default \| primary \| secondary \| accent \| success \| warning \| error \| info \| muted` | `default` | Icon color variant (theme-driven)                     |
| className | `string`                                                                                    | -         | Additional CSS classes                                |

Inherits all standard HTML span attributes (`onClick`, `aria-label`, `aria-hidden`, etc.).

## Accessibility

- Uses `currentColor` for flexible theming
- Supports `aria-hidden` for decorative icons
- Accepts `aria-label` for standalone icons that convey meaning
- Proper semantic HTML structure with `<span>` wrapper
- Consistent sizing ensures minimum touch targets when interactive

## Component API

| Export    | Type                              | Description                        |
| --------- | --------------------------------- | ---------------------------------- |
| Icon      | `React.ForwardRefExoticComponent` | Main component with ref forwarding |
| IconProps | `interface`                       | Component props interface          |
