# Avatar

A versatile avatar component for displaying user profile images with fallback placeholders, status indicators, and multiple shape options.

## Installation & Integration

### Import

```tsx
import { Avatar } from "dandy-ui";
```

## Props

| Prop           | Type                                                   | Default        | Description                            |
| -------------- | ------------------------------------------------------ | -------------- | -------------------------------------- |
| src            | `string`                                               | -              | Image source URL                       |
| alt            | `string`                                               | `""`           | Alt text for accessibility             |
| size           | `xs \| sm \| md \| lg \| xl \| 2xl`                    | `md`           | Avatar size                            |
| shape          | `circle \| rounded \| square`                          | `circle`       | Avatar shape                           |
| outlined       | `boolean`                                              | `false`        | Show border around avatar              |
| placeholder    | `icon \| string`                                       | -              | Placeholder content (icon or initials) |
| status         | `online \| offline \| away`                            | -              | Status indicator                       |
| statusPosition | `top-left \| top-right \| bottom-left \| bottom-right` | `bottom-right` | Position of status indicator           |
| onError        | `() => void`                                           | -              | Callback when image fails to load      |
| className      | `string`                                               | -              | Additional CSS classes                 |

Inherits all standard HTML `div` attributes.

## Accessibility

- Always provide descriptive `alt` text for screen readers
- Status indicators include `aria-label` with current status
- Status indicators use `role="status"` for live region announcements
- Icon placeholders marked with `aria-hidden="true"`
- Initials placeholders include descriptive `aria-label`
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
