# Chips

A controlled input component for managing multiple tag or chip values. Supports keyboard shortcuts (Enter, comma), paste operations, validation, and duplicate detection.

## Installation & Integration

### Import

```tsx
import { Chips } from "dandy-ui";
```

## Usage

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <ComponentPreview>
     <ChipsChange chips={["React", "TypeScript"]} inputValue="" placeholder="Add tags..." />
    </ComponentPreview>
  </Tab>
  <Tab value="Code">
    ```tsx
    const [chips, setChips] = React.useState(["React", "TypeScript"]);
    const [inputValue, setInputValue] = React.useState("");

    <Chips
      chips={chips}
      inputValue={inputValue}
      onInputChange={setInputValue}
      onAddChip={(chip) => setChips([...chips, chip])}
      onRemoveChip={(_, index) => setChips(chips.filter((_, i) => i !== index))}
      placeholder="Add tags..."
    />
    ```

  </Tab>
</Tabs>

## Props

| Prop            | Type                                                                     | Default   | Description                                      |
| --------------- | ------------------------------------------------------------------------ | --------- | ------------------------------------------------ |
| chips           | `string[]`                                                               | -         | Current chips to display (controlled)            |
| inputValue      | `string`                                                                 | -         | Controlled input value                           |
| onInputChange   | `(value: string) => void`                                                | -         | Change handler for the input value               |
| onAddChip       | `(chip: string) => void`                                                 | -         | Add handler called after validation              |
| onRemoveChip    | `(chip: string, index: number) => void`                                  | -         | Remove handler                                   |
| allowDuplicates | `boolean`                                                                | `false`   | Allow duplicate chip values                      |
| maxChips        | `number`                                                                 | -         | Optional cap on number of chips                  |
| chipVariant     | `default \| primary \| secondary \| success \| warning \| error \| info` | `primary` | Badge variant for chips                          |
| chipAppearance  | `outline \| filled`                                                      | `outline` | Badge appearance for chips                       |
| chipSize        | `xs \| sm \| md \| lg`                                                   | `sm`      | Badge size for chips                             |
| inputVariant    | `default \| success \| error \| warning \| info`                         | `default` | Input variant for validation feedback            |
| errorMessage    | `string`                                                                 | -         | Error message to display                         |
| rounded         | `sm \| md \| lg \| xl \| full \| none`                                   | -         | Border radius                                    |
| disabled        | `boolean`                                                                | `false`   | Disables the component                           |
| placeholder     | `string`                                                                 | -         | Placeholder text for input                       |
| onKeyDown       | `React.KeyboardEventHandler<HTMLInputElement>`                           | -         | Optional keydown handler (called after built-in) |
| onPaste         | `React.ClipboardEventHandler<HTMLInputElement>`                          | -         | Optional paste handler (overrides default)       |
| className       | `string`                                                                 | -         | Additional CSS classes                           |

Inherits all standard HTML input attributes (`id`, `name`, `autoComplete`, etc.).

## Accessibility

- Keyboard navigable with Tab key
- Press Enter or comma to add chips
- Click X button or use keyboard to remove chips
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
- Screen reader compatible with proper ARIA labels
- Minimum 44x44px touch targets for mobile
- Error states announced to assistive technologies via InfoTip

## Keyboard Shortcuts

- **Enter**: Add current input value as a chip
- **Comma (,)**: Add current input value as a chip
- **Tab**: Navigate to next focusable element
- **Click X**: Remove individual chip

## Paste Behavior

The component automatically splits pasted content by commas and newlines, allowing users to quickly add multiple chips at once. To override this behavior, provide a custom `onPaste` handler.
