# Accordion

A vertically stacked set of collapsible sections that allow users to show and hide content. Supports single or multiple open panels.

## Installation & Integration

### Import

```tsx
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent,
} from "dandy-ui";
import "dandy-ui/theme.css";
```

## Controlled State

Control the accordion state externally using the `value` and `onValueChange` props.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <ComponentPreview>
      <Accordion type="single" value="item-2">
        <AccordionItem value="item-1">
          <AccordionTrigger>First Item</AccordionTrigger>
          <AccordionContent>This item is closed.</AccordionContent>
        </AccordionItem>
        <AccordionItem value="item-2">
          <AccordionTrigger>Second Item (Controlled Open)</AccordionTrigger>
          <AccordionContent>
            This item is controlled to be open via the value prop.
          </AccordionContent>
        </AccordionItem>
        <AccordionItem value="item-3">
          <AccordionTrigger>Third Item</AccordionTrigger>
          <AccordionContent>This item is closed.</AccordionContent>
        </AccordionItem>
      </Accordion>
    </ComponentPreview>
  </Tab>
  <Tab value="Code">
    ```tsx
    const [value, setValue] = useState("item-2");

    <Accordion type="single" value={value} onValueChange={setValue}>
      <AccordionItem value="item-1">
        <AccordionTrigger>First Item</AccordionTrigger>
        <AccordionContent>This item is closed.</AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-2">
        <AccordionTrigger>Second Item (Controlled Open)</AccordionTrigger>
        <AccordionContent>
          This item is controlled to be open via the value prop.
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-3">
        <AccordionTrigger>Third Item</AccordionTrigger>
        <AccordionContent>This item is closed.</AccordionContent>
      </AccordionItem>
    ```

## Props

### Accordion

| Prop          | Type                                  | Default  | Description                                              |
| ------------- | ------------------------------------- | -------- | -------------------------------------------------------- |
| type          | `single \| multiple`                  | `single` | Whether one or multiple items can be open                |
| value         | `string \| string[]`                  | -        | Controlled value (string for single, array for multiple) |
| defaultValue  | `string \| string[]`                  | -        | Default value for uncontrolled usage                     |
| onValueChange | `(value: string \| string[]) => void` | -        | Callback when value changes                              |
| disabled      | `boolean`                             | `false`  | Disables all accordion items                             |
| collapsible   | `boolean`                             | `false`  | Allow closing all items (only for single type)           |
| className     | `string`                              | -        | Additional CSS classes                                   |

Inherits all standard HTML `div` attributes.

### AccordionItem

| Prop      | Type      | Default | Description                    |
| --------- | --------- | ------- | ------------------------------ |
| value     | `string`  | -       | Unique identifier for the item |
| disabled  | `boolean` | `false` | Disables this specific item    |
| className | `string`  | -       | Additional CSS classes         |

Inherits all standard HTML `div` attributes.

### AccordionTrigger

| Prop      | Type              | Default | Description              |
| --------- | ----------------- | ------- | ------------------------ |
| icon      | `React.ReactNode` | -       | Custom icon to display   |
| hideIcon  | `boolean`         | `false` | Hide the icon completely |
| className | `string`          | -       | Additional CSS classes   |

Inherits all standard HTML `button` attributes.

### AccordionContent

| Prop      | Type     | Default | Description            |
| --------- | -------- | ------- | ---------------------- |
| className | `string` | -       | Additional CSS classes |

Inherits all standard HTML `div` attributes.

## Accessibility

- Keyboard navigable with Tab, Enter, and Space keys
- Focus visible with ring indicator
- WCAG 2.1 Level AA compliant
- Minimum 44px touch targets for mobile
- Proper ARIA attributes (`aria-expanded`, `aria-disabled`, `aria-hidden`)
- Semantic HTML with `h3` elements for triggers
- Screen reader compatible with contextual announcements

## Component API

| Export                | Type        | Description                          |
| --------------------- | ----------- | ------------------------------------ |
| Accordion             | `React.FC`  | Root accordion container component   |
| AccordionItem         | `React.FC`  | Individual accordion section         |
| AccordionTrigger      | `React.FC`  | Clickable trigger for accordion item |
| AccordionContent      | `React.FC`  | Collapsible content area             |
| AccordionProps        | `interface` | Props for Accordion component        |
| AccordionItemProps    | `interface` | Props for AccordionItem component    |
| AccordionTriggerProps | `interface` | Props for AccordionTrigger component |
| AccordionContentProps | `type`      | Props for AccordionContent component |
