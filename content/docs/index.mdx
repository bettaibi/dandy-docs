# Dandy UI

A production-grade, mobile-first React component library built on Tailwind CSS v4. Optimized for edge-first environments like Cloudflare Workers and serverless platforms.

## Key Features

- **Edge-First** — Minimal runtime, optimized for Cloudflare Workers and serverless
- **Theme-Aware** — CSS token system with built-in dark mode support
- **Mobile-First** — Responsive design with 44px touch targets
- **Accessible** — WCAG 2.1 Level AA compliant with keyboard navigation
- **Tree-Shakable** — Import only what you need
- **SSR Compatible** — Works with Next.js, Remix, and any React server framework
