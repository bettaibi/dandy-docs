# Switch

A mobile-first toggle switch component for binary on/off states. Features smooth animations, multiple color variants, and full keyboard and screen reader support.

## Installation & Integration

### Import


## Props

| Prop            | Type                                             | Default   | Description                              |
| --------------- | ------------------------------------------------ | --------- | ---------------------------------------- |
| size            | `sm \| md \| lg`                                 | `md`      | Visual size of the switch                |
| variant         | `default \| success \| warning \| error \| info` | `default` | Color variant when checked               |
| disabled        | `boolean`                                        | `false`   | Disables the switch                      |
| onCheckedChange | `(checked: boolean) => void`                     | -         | Callback fired when toggle state changes |
| className       | `string`                                         | -         | Additional CSS classes                   |

Inherits all standard HTML input[type="checkbox"] attributes (`checked`, `defaultChecked`, `onChange`, `aria-label`, etc.).

## Accessibility

- Keyboard navigable with Tab and Space/Enter keys
- Focus visible with ring indicator
- Uses native checkbox input for proper form submission
- Supports `aria-label` or `aria-labelledby` for screen readers when used without visible label
- WCAG 2.1 Level AA compliant
- Touch-friendly targets (minimum 36px height on small, 44px on medium/large)
- RTL (right-to-left) language support
