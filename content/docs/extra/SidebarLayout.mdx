# SidebarLayout

> This component is part of the `dandy-ui/extra` package. For optimal bundle size, use dynamic imports or lazy loading.

A compound layout component that pairs a sliding sidebar panel with a main content area. Supports push mode (content shifts aside) and overlay mode (sidebar floats over content). Includes a standalone `SideBar` component for simpler use cases.

## Installation & Integration

### Import

```tsx
import { SidebarLayout, SideBar } from "dandy-ui/extra";
```

## Usage

## Props

### SidebarLayout (Root)

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| mode | `"push" \| "overlay"` | `"overlay"` | Display mode |
| placement | `"left" \| "right"` | `"left"` | Sidebar placement |
| layout | `"inner" \| "outer"` | `"outer"` | Fixed to viewport or absolute within parent |
| open | `boolean` | `false` | Controlled open state |
| onOpenChange | `(open: boolean) => void` | - | Callback on open state change |
| width | `"64" \| "72" \| "80" \| "96" \| "full"` | `"80"` | Sidebar width |
| overlay | `"blur" \| "transparent" \| "none"` | `"transparent"` | Backdrop overlay style |
| hidden | `"sm" \| "md" \| "lg" \| "xl"` | - | Hide sidebar above this breakpoint |
| ariaLabel | `string` | `"Sidebar"` | Accessible label |

### SideBar (Standalone)

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| open | `boolean` | `false` | Controlled open state |
| onOpenChange | `(open: boolean) => void` | - | Callback on backdrop click |
| placement | `"left" \| "right"` | `"left"` | Sidebar placement |
| width | `"64" \| "72" \| "80" \| "96" \| "full"` | `"80"` | Sidebar width |
| overlay | `"blur" \| "transparent" \| "none"` | `"transparent"` | Backdrop overlay style |
| layout | `"inner" \| "outer"` | `"outer"` | Positioning behavior |
| hidden | `"sm" \| "md" \| "lg" \| "xl"` | - | Hide above this breakpoint |
| ariaLabel | `string` | `"Sidebar"` | Accessible label |

## Accessibility

- Sidebar uses `role="complementary"` with `aria-label`
- `aria-hidden` toggles based on open state
- `aria-modal` set for overlay mode
- Backdrop click to close
- Content marked `aria-hidden` when overlay sidebar is open
- Smooth transition animations respecting `prefers-reduced-motion`
- Responsive hide at configurable breakpoints
