# DateRangePicker

> This component is part of the `dandy-ui/extra` package. For optimal bundle size, use dynamic imports or lazy loading.

A date range picker that combines an input field with a dual-month calendar popup. Uses a Popover on desktop and a Drawer on mobile. Supports locale-aware formatting, validation, and calendar color variants.

The library also exports a `MultiDatePicker` for selecting multiple individual dates.

## Installation & Integration

### Import

```tsx
import { DateRangePicker } from "dandy-ui/extra";
```

### Peer Dependencies

```bash
npm install react-day-picker
```

When using the `errorMessage` prop, also install:

```bash
npm install @radix-ui/react-tooltip
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| value | `DateRange` | - | Selected date range (controlled) |
| onChange | `(range: DateRange \| undefined) => void` | - | Callback when range changes |
| placeholder | `string` | `"Pick a date range"` | Input placeholder text |
| dateLocale | `"en-US" \| "fr-FR" \| "es-ES" \| "de-DE" \| "it-IT" \| "ar-EG" \| "ar-TN"` | `"en-US"` | Locale for date formatting |
| dateFormat | `Intl.DateTimeFormatOptions` | `{ year: "numeric", month: "short", day: "numeric" }` | Date display format |
| inputSize | `InputProps["size"]` | `"md"` | Input height and padding |
| inputVariant | `InputProps["variant"]` | `"default"` | Input border and focus color |
| errorMessage | `string` | - | Validation error message |
| disabled | `boolean` | `false` | Disables the picker |
| inputClassName | `string` | - | Additional classes for the input |
| contentClassName | `string` | - | Additional classes for the popup |
| align | `"start" \| "center" \| "end"` | `"start"` | Popover alignment (desktop) |
| variant | `"primary" \| "secondary" \| "success" \| "info" \| "warning" \| "error"` | `"primary"` | Calendar color variant |

Also inherits `BaseCalendarProps` (excluding `mode`, `selected`, `onSelect`). Displays 2 months by default.

## Accessibility

- Keyboard navigable: `Enter`/`Space` toggle, `ArrowDown` opens, `Escape` closes
- Calendar keyboard navigation via react-day-picker
- Responsive: uses Popover on desktop, Drawer on mobile
- Read-only input prevents manual text entry errors
- Screen reader accessible with proper ARIA labels
