# Drawer

> This component is part of the `dandy-ui/extra` package. For optimal bundle size, use dynamic imports or lazy loading.

A swipe-able drawer panel built on Vaul. Slides in from any edge of the screen with gesture support, overlay backdrop, and touch-friendly interactions. Ideal for mobile navigation, forms, and contextual panels.

## Installation & Integration

### Import

```tsx
import { Drawer } from "dandy-ui/extra";
```

### Peer Dependencies

```bash
npm install vaul
```

## Show Handle

The drag handle indicator is shown by default on top/bottom drawers. Disable it with `showHandle={false}`.

## Props

### Drawer (Root)

The root component accepts all Vaul `Drawer.Root` props including:

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| direction | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Edge the drawer slides from |
| open | `boolean` | - | Controlled open state |
| onOpenChange | `(open: boolean) => void` | - | Callback on open state change |

### Drawer.Content

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| showHandle | `boolean` | `true` | Show the drag handle indicator |
| className | `string` | - | Additional CSS classes |

Inherits all Vaul `Drawer.Content` attributes.

## Accessibility

- Built on Vaul for full accessibility support
- Focus trap when open, focus returns to trigger on close
- `Escape` key closes the drawer
- Swipe gestures for mobile dismissal
- Background scroll prevention
- Proper ARIA attributes (`aria-modal`, `aria-label`)
- Respects `prefers-reduced-motion`

## Component API

| Export | Type | Description |
| --- | --- | --- |
| Drawer | Compound component | Root drawer controller |
| Drawer.Trigger | `Vaul DrawerTrigger` | Element that opens the drawer |
| Drawer.Content | `React.ForwardRefExoticComponent` | Drawer content container |
| Drawer.Close | `Vaul DrawerClose` | Element that closes the drawer |
| Drawer.Header | `React.ForwardRefExoticComponent` | Header layout container |
| Drawer.Footer | `React.ForwardRefExoticComponent` | Footer layout container |
| Drawer.Title | `React.ForwardRefExoticComponent` | Drawer title heading |
| Drawer.Description | `React.ForwardRefExoticComponent` | Drawer description text |
| DrawerContentProps | `interface` | Props for Drawer.Content |
