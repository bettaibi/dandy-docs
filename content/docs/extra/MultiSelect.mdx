# MultiSelect

> This component is part of the `dandy-ui/extra` package. For optimal bundle size, use dynamic imports or lazy loading.

A compound multi-selection dropdown built on Radix UI Popover. Supports checkboxes, item grouping, labels, separators, max selection limits, and theme-aware variants. Fully keyboard navigable.

## Installation & Integration

### Import

```tsx
import { MultiSelect } from "dandy-ui/extra";
```

### Peer Dependencies

When using the `errorMessage` prop, install the tooltip peer dependency:

```bash
npm install @radix-ui/react-tooltip
```

## Props

### MultiSelect (Root)

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | `"default" \| "success" \| "error" \| "warning" \| "info"` | `"default"` | Visual color variant |
| value | `string[]` | - | Controlled selected values |
| defaultValue | `string[]` | `[]` | Default selected values (uncontrolled) |
| onValueChange | `(values: string[]) => void` | - | Callback when selection changes |
| maxSelection | `number` | - | Maximum selectable items |
| open | `boolean` | - | Controlled open state |
| onOpenChange | `(open: boolean) => void` | - | Callback on open state change |

### MultiSelect.Trigger

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| size | `"sm" \| "md" \| "lg"` | `"md"` | Trigger button size |
| placeholder | `string` | `"Select items"` | Placeholder when nothing selected |
| leftSlot | `React.ReactNode` | - | Content on the left side |
| errorMessage | `string` | - | Error tooltip message |
| rounded | `"sm" \| "md" \| "lg" \| "xl"` | - | Border radius |

### MultiSelect.Item

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| value | `string` | required | Unique value for the item |
| disabled | `boolean` | `false` | Whether the item is disabled |
| leftSlot | `React.ReactNode` | - | Content on the left side |
| rightSlot | `React.ReactNode` | - | Content on the right side |

## Accessibility

- Full keyboard navigation: `ArrowUp`/`ArrowDown`, `Home`/`End`, `Space`/`Enter` to toggle
- `aria-multiselectable="true"` listbox role
- Items use `role="option"` with `aria-selected` and `aria-disabled`
- Focus trap within the dropdown
- Min 44px touch targets for mobile
- Respects `prefers-reduced-motion`
