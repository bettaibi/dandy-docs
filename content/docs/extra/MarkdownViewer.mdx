# MarkdownViewer

> This component is from the `dandy-ui/markdown` package. For optimal bundle size, use dynamic imports or lazy loading.

A full-featured markdown renderer built on `react-markdown`. Supports GitHub Flavored Markdown (tables, task lists, strikethrough), syntax-highlighted code blocks with copy buttons, raw HTML rendering, and XSS sanitization. All typography is theme-aware and responsive.

## Installation & Integration

### Import

```tsx
import { MarkdownViewer } from "dandy-ui/markdown";
import "dandy-ui/syntax-highlighting.css";
```

### Peer Dependencies

```bash
npm install react-markdown rehype-highlight rehype-raw rehype-sanitize remark-gfm
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| content | `string` | required | Markdown content to render |
| gfm | `boolean` | `true` | Enable GitHub Flavored Markdown |
| syntaxHighlighting | `boolean` | `true` | Enable syntax highlighting for code blocks |
| sanitize | `boolean` | `true` | Enable HTML sanitization for XSS protection |
| allowRawHtml | `boolean` | `false` | Allow raw HTML in markdown |
| showCodeCopy | `boolean` | `true` | Show copy button on code blocks |
| components | `Record<string, React.ComponentType>` | - | Custom component overrides |
| className | `string` | - | Additional CSS classes |

Inherits all standard HTML `div` attributes.

## Accessibility

- Semantic HTML output (headings, lists, links, blockquotes)
- Code blocks have a labeled copy button
- Links open in new tabs with `rel="noopener noreferrer"`
- Images include `alt` attributes and lazy loading
- Theme-aware colors adapt to light/dark mode
- Word break applied to prevent overflow
